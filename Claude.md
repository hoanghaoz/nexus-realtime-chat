Role: Chuyên gia Lập trình Frontend
Project: Nexus_RealtimeChatApp (Dự án ứng dụng chat realtime dựa trên dự án mẫu có sẵn nằm trong Folder Moji_RealtimeChatApp)

[YÊU CẦU TỐI THƯỢNG]
Tuyệt đối KHÔNG thay đổi, chỉnh sửa hay can thiệp vào bất kỳ file/cấu trúc nào thuộc thư mục Backend. Nhiệm vụ của bạn chỉ giới hạn ở phạm vi Frontend.

[NHIỆM VỤ CHÍNH]
Thực hiện nối các API đã có sẵn từ Backend vào UI Frontend để hoàn thiện danh sách các tính năng sau:

Xác thực: Đăng ký, Đăng nhập.

Người dùng: Hiển thị thông tin chi tiết user, Tìm kiếm user, Cập nhật thông tin cá nhân của user.

Bạn bè: Lấy danh sách bạn bè của user, Gửi yêu cầu kết bạn, Đồng ý kết bạn, Từ chối kết bạn.

Hội thoại & Chat: Lấy danh sách các cuộc hội thoại của user (để hiển thị), Thực hiện tính năng nhắn tin chat.

Nhóm: Tạo nhóm, Sửa thông tin nhóm, Xóa nhóm, Thêm thành viên vào nhóm.

[QUY TẮC LẬP TRÌNH & XỬ LÝ GIAO DIỆN]

Tuân thủ kiến trúc: Việc nối API phải tuân theo chính xác cấu trúc/kiến trúc tổ chức code của dự án mẫu (cách chia folder services, call API, quản lý state...). Nếu phát hiện luồng nối API nào không đúng chuẩn, bạn có trách nhiệm sửa lại cho khớp với mẫu.

Xử lý UI thiếu: Trong trường hợp đã có API nhưng chưa có UI cho tính năng đó, bạn được quyền tạo mới UI. Tuy nhiên, UI mới phải được xây dựng và tái sử dụng các components dựa trên phong cách thiết kế/UI đã có sẵn ở thư mục dự án mẫu.

Bắt đầu bằng việc rà soát các API cần thiết cho tính năng Đăng ký/Đăng nhập và cung cấp mã nguồn Frontend tương ứng để tích hợp.
